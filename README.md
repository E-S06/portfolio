# Emmanuel Solademi — Portfolio Website

Personal portfolio website for Emmanuel Solademi, a freelance Web Designer & Developer specializing in high-conversion landing pages, brand experiences, and production-ready WordPress builds.

🌐 **Live site:**https://e-s06.github.io/portfolio/

---

## About

This is a single-page portfolio built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies (except Google Fonts). Fully responsive across desktop, tablet, and mobile.

## Features

- Animated particle background (Canvas API)
- Scroll-triggered fade-up animations
- Hamburger navigation menu for mobile
- Projects showcase with tech stack badges
- Skills grid covering design, development, integrations, and workflow
- Contact section with direct email link

## Projects Featured

- **Green Harbs — Maxherb DIM Capsule** · WordPress/Elementor client landing page with Facebook Pixel, FormLayer, and custom JS
- **Crypt-12** · AI-powered crypto platform built in Framer
- **ProlFix** · Premium scroll animation demo using GSAP, Canvas, and Three.js

## Tech Stack

```
HTML5 · CSS3 · Vanilla JavaScript · Canvas API · Google Fonts (Syne + Inter)
```

## Usage

No build step needed. Just open `index.html` in any browser, or host it anywhere — GitHub Pages, Netlify, etc.
